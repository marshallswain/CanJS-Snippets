CanJS-Snippets
==============

Snippets for working with CanJS. http://canjs.com

View the repository to see what is available: https://github.com/marshallswain/CanJS-Snippets

Most of these snippets can be accessed inside .js files.  A few of them are to be used in html or stache templates. (Install the `HTML with Mustaches` package for great highlighting of Stache templates.)

Access the snippets by typing `can-`


Snippets are included for the following parts of CanJS:

## Component Modlet Workflow
See http://blog.bitovi.com/modlet-workflows/
Includes ES6 import syntax for working with StealJS

## Can.Model
See http://canjs.com/docs/can.Model.html

## Can.Route
See http://canjs.com/docs/can.route.html

## Can.View
can.view.tag: http://canjs.com/docs/can.view.tag.html
can.view.attr: http://canjs.com/docs/can.view.attr.html

## Can.Import
See http://canjs.com/docs/can.import.html

## Can.Util methods
See http://canjs.com/docs/can.util.html
